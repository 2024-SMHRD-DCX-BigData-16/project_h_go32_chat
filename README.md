# 📎 TravieAI(팀명: Go32)


## 👀 서비스 소개
* 서비스명: 여행자들을 위한 일정 추천 및 관리 챗봇 서비스
* 서비스설명: 여행 계획 수립부터 실시간 문제 해결까지 AI기반 챗봇으로 한번에 해결하는 스마트 여행 컨시어지 서비스입니다.
<br>

## 📅 프로젝트 기간
2025.02.18 ~ 2025.03.05 (3주)
<br>

## ⭐ 주요 기능
* 여행지 정보 탐색
* 실시간 날짜, 환율, 날씨
* 24시간 챗봇 서비스
* 일정관리
* 사용자 맞춤 여행지 추천
* 대화 맥락을 유지한 개인화된 답변
<br>
